## Start HTTP JSON-RPC

You can start the HTTP JSON-RPC with the --rpc flag
```bash
## mainnet
geth attach https://bsc-dataseed1.binance.org

## testnet
geth attach https://data-seed-prebsc-1-s1.binance.org:8545/
```
